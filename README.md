### Portfolio Generator

## Description

An application that dynamically generates a professional portfolio using node through the command line. 

## Installation
1- Clone repo
2- install NPM and inquirer
3- run app.js

## Usage

1- Run node app.js
2- respond to CLI prompt 
3- Portfolio Generates
